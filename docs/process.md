# Request & Delivery Process

This guide outlines how your request for an API key moves from submission to secure delivery. 

> **API key security is essential at every step.**

## Four Simple Steps

1. **Submit:** Complete the API Key Request Form (HarvardKey required)
2. **Review:** The Academic Technology team reviews your request (usually within 2â€“3 business days)
3. **Receive:** Your API key and starter notebook link will be delivered via **encrypted Harvard email or secure file transfer**
4. **Start:** Begin your project!

### Quick Reference Table

| Step | You Do | We Do | Result |
| :---: | :--- | :--- | :--- |
| 1 | Submit form w/ project details (HarvardKey) | Review, approve, track | Request queued |
| 2 | Wait | Provision key in API portal | Key created |
| 3 | Save your key & link from secure email / file transfer we send | Store key securely and share other keys via secure delivery | **Access granted** |
| 4 | Complete starter notebook and begin your project | Close ticket | Academic work begins |
