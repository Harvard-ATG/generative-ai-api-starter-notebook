# Request & Delivery Process

This guide outlines how your request for an API key moves from submission to secure delivery. 

> **API key security is essential at every step.**

## Four Simple Steps

1. **Submit:** Complete the API Key Request Form (HarvardKey required)
2. **Review:** The Academic Technology team reviews your request (usually within 2â€“3 business days)
3. **Receive:** Your API key and starter notebook link will be delivered via **encrypted Harvard email or secure file transfer**
4. **Start:** Begin your project!

## Security Reminders

- **Never** post/share your API key publicly (Slack, GitHub, email, etc.)
- Your API Key should be treated like a sensitive password. Follow [Harvard's Information Security and Data Privacy - Protect Identity: Best Practices](https://privsec.harvard.edu/protect-identity) (including the use of Password Managers) to keep your API Key secure.
- If distributing keys for others, do so securely by following Harvard [secure email](https://harvard.service-now.com/ithelp?id=kb_article&sys_id=cea973b3dbf8e0d0441560fdd39619d8) or [secure file transfer](https://harvard.service-now.com/ithelp?id=kb_article&sys_id=3c725c7d47016950566cf147536d43ff) policies.
